# STXWORX

Decentralized Freelance Platform with Milestone-Based Escrow on Stacks Blockchain

## Tech Stack

- React 18
- Vite
- React Router DOM
- GSAP
- Swiper

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Routes

- `/` - Landing (marketing page, no sidebar)
- `/home` - Dashboard (with sidebar)
- `/jobs` - Browse Jobs
- `/subscription` - Pro Subscription
- `/projects` - Active Projects
- `/freelancers` - Browse Freelancers
- `/leaderboard` - Leaderboard
- `/disputes` - Dispute Resolution
- `/how-it-works` - How It Works

## Key Features

- ✅ Full React SPA with routing
- ✅ Responsive design with dark theme
- ✅ Separate Landing page (no sidebar) and Dashboard (with sidebar)
- ✅ Custom animations (GSAP) with hover effects
- ✅ Component-based architecture
- ✅ STXWORX branding throughout
- ✅ Freelancer cards with ratings and hourly rates
- ✅ Project cards with milestones and proposals
- ✅ Colored page titles with muted accent colors
- ✅ Professional navigation and layout
